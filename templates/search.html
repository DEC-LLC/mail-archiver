How to Search

Type a word or phrase in the search box. All other fields are optional filters. The search box is the only required field — everything else narrows your results.

What you want Search box Filters to set
Find all emails about "invoice" invoice (none needed)
Emails from John about the project project From: john@
All emails in March 2025 * From date: 2025-03-01, To date: 2025-03-31
Emails with attachments from Gmail only * Account: (pick your Gmail), Attachments: Has attachments
Exact phrase "quarterly report" (none needed)
Emails containing budget OR forecast budget OR forecast (none needed)
Exclude a term meeting NOT cancelled (none needed)
Search subject lines only important Search in: Subject only
Search email body only password reset Search in: Body only

Tips

  • The search box searches subjects, senders, recipients, and full message body
  • Use quotes for exact phrases: "board meeting"
  • Use OR between words to match either: receipt OR invoice
  • Use NOT to exclude: project NOT cancelled
  • Use * as a wildcard prefix: financ* matches finance, financial, financing
  • Leave the search box empty and just set date range or account to browse recent emails matching those filters
  • Export any search results as MBOX (for email clients) or EML zip (individual files)
