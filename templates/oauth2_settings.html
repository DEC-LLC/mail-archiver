Microsoft OAuth2 Settings

To use Microsoft (Outlook/Hotmail) accounts with OAuth2, you need to register a free Azure AD app. This gives you a Client ID and Client Secret that Mail Archiver uses to authenticate with Microsoft on your behalf.

Setup takes about 5 minutes. You only do this once.
  1. Go to Azure App Registrations
  2. Click New Registration
  3. Name: Mail Archiver
  4. Supported account types: Accounts in any organizational directory and personal Microsoft accounts
  5. Redirect URI: Web → {{ request.url_root }}oauth2/callback
  6. Click Register
  7. Copy the Application (client) ID — paste below
  8. Go to Certificates & secrets → New client secret
  9. Copy the secret Value (not the ID) — paste below
  10. Go to API permissions → Add a permission → APIs my organization uses → search Office 365 Exchange Online → Delegated → IMAP.AccessAsUser.All
